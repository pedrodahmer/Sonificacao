O projeto

Esta aplicação faz parte do projeto realizado para a disciplina de Engenharia de Software II do curso de Informática Biomédica da Universidade Federal de Ciências da Saúde de Porto Alegre (UFCSPA).

A aplicação realiza a sonificação multi-paramétrica (altura, duração e intensidade) a partir de planilhas de dados .csv. O formato de saída é .mid, que vem do padrão MIDI (Musical Instrument Digital Interface), uma linguagem inicialmente voltada para a comunicação entre dispositivos musicais e computadores.

O programa realiza um pré-processamento dos dados a serem sonificados, utilizando-se do programa csvmidi desenvolvido por John Walker para realizar a conversão entre os formatos.
